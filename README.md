\# Network Traffic Analysis Using Wireshark



\## Project Overview



This project demonstrates how network traffic can be captured and analyzed using Wireshark. Various network protocols such as DNS, TCP, TLS, and ICMP were examined to understand communication between devices on a network.



\## Objective



\- Capture live network traffic.

\- Analyze common network protocols.

\- Understand packet structures.

\- Gain practical SOC Analyst skills.



\## Tools Used



\- Wireshark

\- Windows 11

\- Internet Connection



\## Protocols Analyzed



\### DNS

Used to resolve domain names into IP addresses.



\### TCP

Responsible for reliable communication between devices.



\### TLS

Provides encrypted communication over the network.



\### ICMP

Used for connectivity testing and diagnostics.
